# Security Policy

## 安全模型

「钱都去哪了」不提供账号、后端、数据库或云端同步。账单解析、统计、图表和 Excel 导出均在浏览器本地完成；离线版本通过 CSP 禁止网络连接。

- 账单内容不主动上传；
- PDF 打开密码只用于当前内存解析；
- 预算只尝试写入当前浏览器会话，不保存交易；
- Excel 导出会将可能被表格软件识别为公式的文本转成普通文本；
- 发布包只包含白名单内的网页资源，不包含账单、报告或 Python 分析结果。

## 支持范围

当前维护分支为 V3。旧版静态产物或被自行修改的离线文件不保证获得安全修复。

## 报告问题

如果发现可能暴露交易数据、绕过本地处理、执行恶意导入内容或泄露 PDF 密码的问题，请不要在公开 Issue 中附带真实账单、截图或个人信息。请先私下联系项目维护者，提供最小化、脱敏的复现步骤；维护者会确认影响并协调修复与披露时间。
